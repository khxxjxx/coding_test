## 괄호 공장장 체셔

### 문제 설명

- 올바르진 못한 괄호 문자열을 최대한 길게 잘라 올바른 괄호로 변환하는 문제.<br>

### 자료 구조

- N<br>

  - 타입 : 정수
  - 저장 데이터 : 초기의 괄호길이

- B<br>
  - 타입 : 문자열
  - 저장 데이터 : 괄호

### 풀이 과정

```txt
1. 만약 괄호 문자열에 올바른 괄호 '()'가 있다면 while문을 계속해서 회전한다.

2. 올바른 문자열을 지워준다

3. 초기의 괄호길이에서 올바른 문자열이 모두 지워지고 남은 괄호의 길이만큼 빼준다.
```

### 코드 구현

```py
N = int(input())
B = input()

# 풀이 과정 1
while '()' in B :
    # 풀이 과정 2
    B = B.replace('()', '')

# 풀이 과정 3
print(N - len(B))
```

input

```
5
(())(
```

output

```
4
```
