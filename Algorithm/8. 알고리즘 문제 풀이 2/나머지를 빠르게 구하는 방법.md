## 나머지를 빠르게 구하는 방법

### 문제 설명

- 첫째줄에 A, 둘쩨줄에 x가 주어질 때, A의 x제곱을 1,000,007로 나눈 나머지를 출력하는 문제.<br>

### 자료 구조

### 풀이 과정

```txt
1. pow함수를 이용하여 세번째 인자로 나눌 수를 입력하면 시간초과되지 않고 빠르게 답을 구할 수 있다.
```

### 코드 구현

사용 언어 : 파이썬

```py
# 풀이 과정 1
print(pow(int(input()), int(input()), 1000007))
```

input

```
100
100
```

output

```
364467
```

점수 : 100 / 100<br>
